# Experiment Templates

## Core Template
- Research Question
- Hypothesis
- Variables (independent, dependent, controls)
- Methods (detailed, reproducible)
- Data Collection Plan (sampling, instrumentation)
- Analysis Plan (statistical methods, software)
- Expected Outcomes
- Limitations and Assumptions
- Ethical Considerations (IRB, consent)

## Domain Adaptation
- For biomechanics: include sensor placement, calibration, movement protocols
- For computer vision: include dataset curation, annotation protocols
- For rehabilitation: include patient criteria, intervention protocols
- For neuroperformance: include cognitive/behavioral tasks, neuroimaging setup
