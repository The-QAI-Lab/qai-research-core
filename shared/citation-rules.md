# Citation Rules

All skills must ensure academic credibility and traceability by:
- Using a consistent, explicit citation format (APA 7th or domain standard)
- Including DOIs or persistent URLs for all sources
- Listing all sources in a dedicated Citations section, matching in-text markers
- Using in-text citation markers (e.g., [1], [2]) that correspond to the Citations section
- Citing all claims, data, and direct quotations
- Reporting missing or uncertain sources explicitly

## Example
- [1] Smith, J. (2020). Title. *Journal*, 12(3), 45-67. https://doi.org/10.xxxx/xxxx
