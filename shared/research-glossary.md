# Research Glossary

- **FAIR:** Findable, Accessible, Interoperable, Reusable (data principles)
- **IRB:** Institutional Review Board (ethics oversight)
- **Protocol:** Detailed, reproducible plan for a scientific experiment
- **Feature Engineering:** Systematic creation of input variables for analysis or modeling
- **QA:** Quality Assurance (systematic review of data/process integrity)
- **Schema:** Structured, explicit definition of data fields and metadata
- **Upstream/Downstream:** Skills or processes that provide inputs to, or consume outputs from, another skill
- **Output Contract:** Explicit, standards-driven structure for skill outputs
