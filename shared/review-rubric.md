# Review Rubric

Use this rubric to evaluate skills, outputs, and research artifacts. All criteria must be addressed in review reports.

| Criterion        | Description                                                      |
|------------------|------------------------------------------------------------------|
| Clarity          | Are instructions, logic, and outputs explicit and unambiguous?   |
| Rigor            | Is scientific rigor and methodological soundness maintained?     |
| Reproducibility  | Can results and outputs be independently reproduced?             |
| Domain Fit       | Is the output relevant and credible for the target research area?|
| Transparency     | Are methods, assumptions, and limitations fully explicit?        |
| Structure        | Are outputs well-organized, labeled, and standards-compliant?    |
| Traceability     | Are all claims and data points properly cited and referenced?    |
| Failure Handling | Are errors, limitations, and uncertainties reported explicitly?  |
