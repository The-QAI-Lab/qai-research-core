# Literature Review Skill — Behavioral Validation Tests

- Does the skill produce a structured summary with explicit section headers and evidence clusters?
- Are consensus/disagreement and evidence strength surfaced and mapped?
- Are all claims traceable to citations (APA 7th or domain standard)?
- Are limitations, uncertainties, and assumptions explicitly stated in dedicated sections?
- Does the output include a Gap Handoff Block suitable for direct use by Gap Finder and Paper Outliner?
- Does the output avoid generic, unsupported, or unverifiable statements?
- Is the output format compliant with `standards/output-contracts.md`?
- Does the skill request clarification if the input is ambiguous?
- Does it handle errors, missing data, and ambiguous evidence gracefully, reporting them explicitly?
- Does the skill distinguish evidence-backed claims from inference or recommendation?
