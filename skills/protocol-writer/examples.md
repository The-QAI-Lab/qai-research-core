# Protocol Writer Skill — Examples

## Example 1: Generic Academic
**Input:** Write a protocol for a double-blind drug trial (adult population, 2024, CONSORT compliance required). Include step-to-design rationale mapping, checkpoints, deviations tracking, and Protocol Handoff Block.
**Output:**
- Step-by-step protocol, each step linked to design rationale
- Materials table (drug, placebo, blinding materials)
- Compliance checklist (CONSORT, IRB, safety)
- Checkpoints and deviations tracking section
- Protocol Handoff Block (machine-readable)

## Example 2: Biomechanics
**Input:** Protocol for motion capture study of running gait in collegiate athletes (2023–2024, field setting, IRB required). Include operational failure points, checkpoints, and explicit documentation requirements.
**Output:**
- Detailed steps (participant prep, sensor placement, data collection), each with rationale
- Equipment/materials list (motion capture system, calibration tools)
- IRB compliance notes and checklist
- Checkpoints and deviations tracking
- Documentation requirements section
- Protocol Handoff Block

## Example 3: QAI-Style Applied
**Input:** Protocol for field validation of wearable sensors in soccer (10 sensors, 2024, operational constraints: outdoor use). Include explicit documentation requirements, operational failure points, and handoff block.
**Output:**
- Structured protocol with step-to-design rationale mapping
- Materials table (sensor types, batteries, data loggers)
- Reporting and compliance checklist (operational, ethical)
- Checkpoints and deviations tracking
- Documentation requirements
- Protocol Handoff Block
