# Statistical Analysis Skill — Examples

## Example 1: Generic Academic
**Input:** Plan analysis for a randomized controlled trial (2024, feature list and Feature Handoff Block attached, CONSORT reporting). Include hypothesis-to-test linkage, assumption checks, primary/fallback analysis, and Paper Claims Block.
**Output:**
- Analysis plan (sections: primary/secondary outcomes, covariates, exploratory/confirmatory distinction)
- Statistical test table (test, variables, assumptions, rationale, hypothesis linkage)
- Primary and fallback analysis paths
- Power/sample size notes and interpretation cautions
- Reporting structure (tables, figures, summary statistics)
- Paper Claims Block (machine-readable)

## Example 2: Biomechanics
**Input:** Statistical analysis for knee joint loading study (adult athletes, 2023–2024, repeated measures, feature list and Feature Handoff Block attached). Include assumption checks, exploratory/confirmatory distinction, and Paper Claims Block.
**Output:**
- Explicit tests (e.g., mixed models, ANOVA), rationale, and reporting plan
- Test table (test, variables, assumptions, rationale, hypothesis linkage)
- Exploratory vs. confirmatory analysis distinction
- Power/sample size notes
- Paper Claims Block

## Example 3: QAI-Style Applied
**Input:** Analysis plan for wearable sensor data in soccer (10 sensors, 2024, harmonization with lab data required, feature list and Feature Handoff Block attached). Include fallback analysis, interpretation cautions, and Paper Claims Block.
**Output:**
- Structured plan (sections: harmonization, missing data, sensor errors, exploratory/confirmatory distinction)
- Test table (test, variables, rationale, hypothesis linkage)
- Primary and fallback analysis paths
- Interpretation cautions
- Paper Claims Block
