# Paper Outliner Skill — Examples

## Example 1: Generic Academic
**Input:** Outline a review paper on deep learning in medical imaging (target: Nature Reviews, 2024, IMRAD structure, Paper Claims Block attached).
**Output:**
- Section breakdown (Introduction, Methods, Results, Discussion, Conclusion)
- Section notes (purpose, content, rationale, claims-to-evidence linkage)
- Figure/table strategy (e.g., summary tables, evidence maps)
- Reviewer Exposure Block (weak sections: Discussion, missing evidence: external validation)
- Publication readiness checkpoint (all core sections present, missing external validation)
- References section (placeholder)
