# Experiment Design Skill — Examples

## Example 1: Generic Academic
**Input:** Design a study to evaluate a new image segmentation algorithm for medical imaging (2024, human data, IRB required). Include both a primary and alternative design, tradeoff matrix, risk register, and Protocol + Schema Handoff Block.
**Output:**
- Structured design table (primary: RCT with blinded assessment; alternative: crossover design)
- Tradeoff matrix (sample size, bias risk, feasibility)
- Risk register (IRB delays, data quality)
- Explicit assumptions (e.g., data availability, annotation quality)
- Protocol + Schema Handoff Block (machine-readable)

## Example 2: Biomechanics
**Input:** Plan an experiment to assess knee joint loading during squats in collegiate athletes (2023–2024, field setting). Include confounder analysis, alternative design, and structured handoff.
**Output:**
- Primary design: field-based repeated measures; alternative: lab-based controlled trial
- Tradeoff matrix (ecological validity vs. measurement precision)
- Confounders (athlete experience, fatigue)
- Risk register (equipment failure, compliance)
- Protocol + Schema Handoff Block

## Example 3: QAI-Style Applied
**Input:** Develop an experiment to compare wearable sensor accuracy in field vs. lab settings (adult athletes, 2024, resource constraints: 10 sensors). Include explicit assumptions, tradeoff analysis, and handoff block.
**Output:**
- Structured design table (primary: parallel group; alternative: within-subject)
- Tradeoff matrix (statistical power, logistical complexity)
- Explicit assumptions (sensor calibration, participant adherence)
- Risk register (data loss, sensor malfunction)
- Protocol + Schema Handoff Block
