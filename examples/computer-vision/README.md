# Computer Vision Domain — Example Applications

Research Core skills enable computer vision research by:
- Reviewing literature on image segmentation or object detection
- Finding gaps in explainable AI or dataset bias
- Designing experiments for algorithm benchmarking
- Writing protocols for annotation or data collection
- Defining schemas for image/video datasets
- QA for annotation consistency and data integrity
- Feature engineering for pixel, region, or temporal analysis
- Planning statistical analysis for model performance
- Outlining papers for CVPR/ICCV/NeurIPS
- Reviewer mode for pre-submission checks
