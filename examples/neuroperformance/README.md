# Neuroperformance Domain — Example Applications

Research Core skills enable neuroperformance research by:
- Reviewing literature on cognitive and motor performance
- Identifying gaps in neurofeedback or training protocols
- Designing experiments for brain-computer interface studies
- Writing protocols for EEG or neuroimaging trials
- Defining schemas for neurophysiological data
- QA for EEG, fNIRS, or behavioral datasets
- Feature engineering for cognitive/behavioral metrics
- Planning analysis for performance improvement studies
- Outlining papers for neuroscience journals
- Reviewer mode for pre-submission and QA
