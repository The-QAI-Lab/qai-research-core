# Rehabilitation Domain — Example Applications

Research Core skills support rehabilitation research by:
- Synthesizing literature on therapy interventions
- Identifying gaps in outcome measurement
- Designing experiments for clinical trials
- Writing protocols for patient studies
- Defining schemas for patient-reported outcomes
- QA for clinical and sensor datasets
- Feature engineering for recovery metrics
- Planning analysis for intervention efficacy
- Outlining manuscripts for rehabilitation journals
- Reviewer mode for protocol and manuscript review
